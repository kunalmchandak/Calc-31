# Multifunction Calculator (C)

A powerful, terminal-based **Multifunction Calculator** written in C that offers **31 features** ranging from basic arithmetic to advanced matrix operations, number theory, and utility tools ‚Äî all in a simple menu-driven interface.

---

## How to Compile and Run

### Compilation
```bash
gcc calci.c -o calci -lm
```

### Run
```bash
./calci
```

---
